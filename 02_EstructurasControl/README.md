# Estructuras de Control

Este directorio contiene ejemplos de estructuras de control en C# y Python, organizados por tipo: secuenciales, condicionales y repetitivas. Cada carpeta incluye los ejercicios desarrollados, con código comentado para facilitar su comprensión y ejecución.

## Secuenciales
Contiene programas que se ejecutan de manera lineal, siguiendo un flujo de instrucciones paso a paso.  
Ejemplos incluidos:
- Cálculo de área y perímetro de un terreno rectangular (C# y Python).
- Cálculo del volumen de un cilindro (C# y Python).
- Conversión de metros a pulgadas (C#).
- ## Condicionales
Contiene programas que toman decisiones usando estructuras condicionales:
- **Simples**: Ejercicios que verifican una sola condición.
- **Dobles**: Ejercicios que verifican dos posibles caminos (if-else).
  - Ejemplo: Encontrar el mayor de 3 números (C#).
- **Anidadas**: Condicionales dentro de otras condicionales.
- **Múltiples**: Uso de `switch` o múltiples condiciones.
  - Ejemplo: Impuesto de renta según UITS (C#).
- ## Repetitivas
Contiene ejercicios que repiten instrucciones usando bucles:
- **For**: Ejercicios de conteo y sumatoria (C# y Python).
  - Ejemplo: Sumar números pares entre 100 y 200.
- **While**: Ejercicios que repiten mientras se cumpla una condición.
- **DoWhile**: Ejercicios que garantizan al menos una ejecución.
